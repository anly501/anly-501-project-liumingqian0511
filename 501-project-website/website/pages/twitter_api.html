Data Gathering from Twitter API

I gathered my text data from Twitter API. Click here to access the Python API code. Firstly, I created a list of search words that included 10 keywords related to the project topic. For each keyword in the search list, I gathered 300 tweets contain those keywords and stored them here as json file. Based on the output of WordCloud, I created another list of 10 words that are most frequently shown in tweets with the hashtag "insurance". As you can see from the result in the WordCloud tab, even without spefifying the type of insurance, the results of most referenced words are still health related. I looped through each json file and sorted the keywords by their frequency of reference in tweets. A frequency dataset was created along with a bar plot to visualize some information about the dataset.

As shown in the bar plot, "health", "cover" and "doctor" are the most frequently referenced words besides "insurance" in tweets related to medical insurance. This is just a naive look at the dataset, more analysis will be done on the text.
