Data Gathering from Wikipedia API

Besides tweets, I used Wikipedia API to gather more text data for this project. Firsly, I used the search method of Wikipedia class to generate a list of titles of Wikipedia page relevant to health insurance. Looping through the titles of this list, I pulled information from the pages and stored them individualy to text files and also to an empty string which was created for WordCloud.
